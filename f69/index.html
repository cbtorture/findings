F69: Level-order test for function evaluation


Description

Tests f1 and F2 have demonstrated clearly that strict left-to-right and right-to-left behaviour is adhered to in simple cases whereby there are no nested function calls in the expressions that make up the arguments to the outer function calls.

What this test case aims to show is that the ordering is not only strictly left-to-right or right-to-left but also that it is for when one does nested calls and that the logic of sequence points applies still. Really it just reassures us the logic holds.
